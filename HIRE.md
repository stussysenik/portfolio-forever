## Design Philosophy: Inverse-Law

> "The more you leave out, the more you magnify what remains."

This portfolio follows three mentors:

1. **Rick Rubin** — Essentialism. Strip away everything that doesn't serve the core message. If it doesn't make a recruiter want to schedule a call, it doesn't belong.

2. **PG LANG** — Purposeful boldness. Every element is intentional. No filler. The work speaks; the design amplifies. Fearless creative vision meets technical execution.

3. **Light Phone** — Digital restraint. Calm over chaos. Focus over feature-bloat. Presence over notification anxiety. The interface gets out of the way so the proof can breathe.

## Proof Architecture

The portfolio is structured as evidence, not theater:

- **`/works`** — Live shipped surfaces, not poster frames. Real URLs, real interactions, real code.
- **`/cv`** — Role history with outcomes, not responsibilities. What was built, what changed.
- **`/process`** — How the work gets done. Problem → Constraint → Metric.
- **`/terminal`** — Engineering depth. 30 commands, virtual filesystem, built with the same stack.
- **`/hire`** — The distilled case. 30-second scan for recruiters. Direct evidence mapping.

## The Stack as Proof

This isn't a template. It's a custom operating system:

- **Astro 6** + **Svelte 5** + **React 19** — Modern, hybrid, performant
- **Convex** — Real-time reactive backend (20+ tables)
- **Sanity** — Editorial CMS with live preview
- **Pure CSS + OKLCH** — Design system with perceptual color uniformity
- **TypeScript end-to-end** — Type safety from database to DOM
- **Playwright + Vitest** — Tested, accessible, validated
